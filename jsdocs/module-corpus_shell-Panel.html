CreateNewSearchPanel(configIdx, searchStr)

purpose: creates a new search panel and displays it.

  1. Creates a new div and styles it as needed by the ui. Its id is set to the unique panel id.
  2. Creates the "window title bar" using module:corpus_shell~Panel#GeneratePanelTitle
  3. Builds a query to be executed if some search string is given.
  4. Creates the ui elements for searching.
    -> module:corpus_shell~Panel#GenerateSearchInputs
    -> module:corpus_shell~Panel#GenerateSearchNavigation
  5. Adds a div for search results. -> module:corpus_shell~Panel#GenerateSearchResultsDiv
  6. Makes the panel draggable. -> module:corpus_shell~Panel#InitDraggable
Parameters:
Name Type Description
configIdx number Preselected index of SearchCombo
searchStr string Search string to inserted in search input
