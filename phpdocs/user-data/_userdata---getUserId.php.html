Generates a GUID for identifying a particular corpus_shell user's persistence file on this web server

A utility service which is otherwise unrelated to the storage and retrieval of the data even though the identifier will be used as a part of the filename on the server. The data is presented to the client as a JSON object with a property of "id". The browser is advised to refetch such GUIDs every time.
