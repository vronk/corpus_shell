vlibTemplateDebug

Description
Description | Vars (details) | Methods (details)

Class uses all of vlibTemplate's functionality but debugs the template files.

Outputs a debug console to an html page, only use this for html files.

NB: This class is not as well documented as the main vlibTemplate class as all functions are private. This will be cleaned up in the future but it's quite far down the priority list. It works, and right now, that's what counts.
