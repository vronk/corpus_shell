Stylesheet data2view_tei.xsl
Documentation

Description

Stylesheet for formatting TEI-elements inside a FCS/SRU-result. the TEI-elements are expected without namespace (!) (just local names) This is not nice, but is currently in results like that.

The templates are sorted by TEI-elements they match. if the same transformation applies to multiple elements, it is extracted into own named-template and called from the matching templates. the named templates are at the bottom.
